### 2.1 表单构建器 ✅
**位置**: `/admin/forms`

**功能**:

- `app/api/admin/forms/route.ts` - 表单API
- `app/api/admin/forms/[id]/route.ts` - 表单详情API

**依赖**: `@dnd-kit/core`, `@dnd-kit/sortable`, `@dnd-kit/utilities`

## 📝 使用说明

### 表单构建器
1. 访问 `/admin/forms`
2. 点击"Create New Form"
3. 添加字段（拖拽或点击字段类型）
4. 配置字段验证规则
5. 设置表单提交处理（邮件、Webhook等）
6. 保存表单
7. 使用 `FormRenderer` 组件在前端渲染表单

### 主题定制器
1. 访问 `/admin/settings/theme`
2. 调整颜色、字体、间距等
3. 实时预览效果
4. 保存主题
5. 主题会自动应用到网站

### 图片编辑器
1. 访问 `/admin/images/editor`
2. 选择图片
3. 调整旋转、滤镜、质量
4. 下载编辑后的图片

### 翻译管理
1. 访问 `/admin/translations`
2. 选择语言
3. 编辑翻译内容
4. 使用AI批量翻译
5. 导出翻译文件

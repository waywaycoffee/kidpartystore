## ⚠️ 注意事项

### 当前限制：

1. **Stripe 支付**
   - 需要配置 `NEXT_PUBLIC_STRIPE_SECRET_KEY` 才能使用
   - 当前为示例代码，需要真实 API 密钥

2. **物流 API**
   - 当前为模拟实现
   - 需要配置真实的 DHL/FedEx API 密钥

3. **多语言**
   - 当前使用内联翻译
   - 建议后续迁移到 `public/locales` 文件

4. **购物车持久化**
   - 当前使用 Redux（内存状态）
   - 刷新页面后会丢失
   - 建议添加 localStorage 或服务端同步

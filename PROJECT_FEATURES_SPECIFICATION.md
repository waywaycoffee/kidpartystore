# PartyExpert 项目功能说明书

## 📋 项目概述

**PartyExpert** 是一个基于 Next.js 的儿童派对用品跨境电商平台，提供完整的产品管理、订单处理、客户服务、内容管理和AI驱动的营销功能。

### 技术栈

- **前端框架**: Next.js 14 (App Router)
- **UI框架**: React + Tailwind CSS
- **状态管理**: Redux Toolkit
- **认证**: NextAuth.js
- **国际化**: react-i18next
- **AI集成**: OpenAI API
- **数据存储**: JSON文件（可扩展至数据库）

## 🚀 部署与运维

### 环境要求

- Node.js 18+
- npm/yarn/pnpm
- 环境变量配置

### 环境变量

```env
# 基础配置
NEXT_PUBLIC_SITE_URL=https://yourdomain.com
NEXTAUTH_URL=https://yourdomain.com
NEXTAUTH_SECRET=your_secret_key

# 管理员配置
ADMIN_EMAIL=admin@example.com
ADMIN_PASSWORD=your_password

# 邮件配置
SMTP_HOST=smtp.example.com
SMTP_PORT=587
SMTP_USER=your_email
SMTP_PASS=your_password
SMTP_FROM=noreply@yourdomain.com

# AI配置
OPENAI_API_KEY=your_openai_api_key

# 图片存储（可选）
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret

# 视频配置（可选）
NEXT_PUBLIC_HOMEPAGE_VIDEO_URL=https://youtube.com/watch?v=xxx
```

### 部署方式

- **Vercel**: 推荐部署平台（一键部署）
- **Docker**: 容器化部署
- **传统服务器**: Node.js服务器部署

## 🔄 工作流程

### 订单处理流程

1. 用户添加商品到购物车
2. 用户进入结账页面
3. 填写配送信息
4. 选择支付方式
5. 提交订单
6. 系统生成订单
7. 发送订单确认邮件
8. 管理员处理订单
9. 更新订单状态为"已发货"
10. 发送发货通知邮件
11. 用户收到商品
12. 订单完成

### 弃购挽回流程

1. 用户添加商品到购物车
2. 用户离开网站（未完成购买）
3. 系统自动保存购物车
4. 1小时后自动发送挽回邮件
5. AI生成个性化邮件内容
6. 邮件包含折扣码和恢复链接
7. 用户点击链接恢复购物车
8. 完成购买

# PartyExpert 产品说明书

## 📋 产品概述

**PartyExpert** 是一个面向欧美市场的跨境儿童生日派对用品独立站电商平台，基于 Next.js 14 构建，提供完整的产品管理、订单处理、客户服务、内容管理和AI驱动的营销功能。

### 产品定位

- **目标市场**: 欧美市场（美国、欧洲等）
- **产品类别**: 儿童生日派对用品（气球、装饰品、餐具、主题套餐等）
- **业务模式**: B2C 跨境电商独立站
- **技术架构**: 现代化全栈 Web 应用

### 核心价值

1. **完整的电商解决方案**: 从产品展示到订单完成的全流程支持
2. **AI驱动的营销**: 智能内容生成、个性化邮件、客服机器人
3. **SEO优化**: 内置SEO工具，提升搜索引擎排名
4. **多语言多货币**: 支持国际化，适应跨境业务需求
5. **易于管理**: 直观的管理后台，无需技术背景即可操作

## 📖 使用指南

### 快速开始

#### 1. 环境准备

```bash
# 安装依赖
npm install

# 配置环境变量
cp .env.example .env.local
```

#### 2. 必需配置

在 `.env.local` 中配置：

```env
# NextAuth配置（必需）
NEXTAUTH_URL=http://localhost:3000
NEXTAUTH_SECRET=your-generated-secret-key

# 管理员账号（必需）
ADMIN_EMAIL=admin@yourdomain.com
ADMIN_PASSWORD=your-secure-password
```

生成 `NEXTAUTH_SECRET`:
```bash
node -e "console.log(require('crypto').randomBytes(32).toString('base64'))"
```

#### 3. 启动开发服务器

```bash
npm run dev
```

访问：http://localhost:3000

#### 4. 首次登录

访问：http://localhost:3000/admin/login

使用配置的管理员账号登录。

### 产品配置

#### 添加产品

1. 访问 `/admin/products`
2. 点击"添加产品"
3. 填写产品信息：
   - 基本信息（名称、描述、价格、库存）
   - 上传产品图片（支持多张）
   - 设置产品视频URL（可选）
   - 配置产品变体（可选）
   - 设置产品分类和标签
4. 保存产品

#### 批量导入产品

1. 访问 `/admin/products/import`
2. 下载导入模板
3. 填写产品数据
4. 上传CSV或JSON文件
5. 预览导入数据
6. 确认导入

### 订单处理

#### 订单流程

1. 用户下单后，订单出现在 `/admin/orders`
2. 查看订单详情，确认信息
3. 处理订单：
   - 待付款：等待用户支付
   - 已付款：准备发货
   - 已发货：更新物流信息，系统自动发送发货邮件
   - 已完成：订单完成
   - 已取消：取消订单

#### 添加物流追踪

1. 进入订单详情页
2. 点击"添加追踪号"
3. 输入物流公司和追踪号
4. 保存后，客户可查看物流状态

### AI功能使用

#### AI内容生成

1. 访问 `/admin/ai/content-generator`
2. 输入产品信息
3. 选择目标语言
4. 点击"生成内容"
5. 复制生成的内容到产品描述

#### AI个性化邮件

1. 访问 `/admin/ai/personalization`
2. 输入用户邮箱
3. 选择邮件类型
4. 点击"生成邮件"
5. 查看生成的个性化邮件内容

#### AI SEO关键词挖掘

1. 访问 `/admin/ai/seo-keywords`
2. 输入产品关键词或主题
3. 点击"生成关键词"
4. 查看生成的三类关键词
5. 一键添加到关键词管理系统

#### AI库存预测

1. 访问 `/admin/ai/inventory`
2. 查看AI分析结果：
   - 高需求产品（建议增加库存）
   - 滞销产品（建议清仓）
   - 销售趋势分析
3. 根据建议调整库存

### 弃购挽回

#### 自动挽回

系统会自动：
1. 跟踪用户购物车
2. 用户离开1小时后，自动发送挽回邮件
3. AI生成个性化邮件内容
4. 自动生成折扣码

#### 手动挽回

1. 访问 `/admin/marketing/abandoned-carts`
2. 查看弃购购物车列表
3. 点击"发送邮件"手动发送挽回邮件

## 🔧 部署说明

### 环境要求

- Node.js 18+
- npm/yarn/pnpm
- 环境变量配置

### 环境变量配置

```env
# 基础配置
NEXT_PUBLIC_SITE_URL=https://yourdomain.com
NEXTAUTH_URL=https://yourdomain.com
NEXTAUTH_SECRET=your_secret_key

# 管理员配置
ADMIN_EMAIL=admin@example.com
ADMIN_PASSWORD=your_password

# 邮件配置
SMTP_HOST=smtp.example.com
SMTP_PORT=587
SMTP_USER=your_email
SMTP_PASS=your_password
SMTP_FROM=noreply@yourdomain.com

# AI配置
OPENAI_API_KEY=your_openai_api_key

# Stripe配置（支付）
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_live_...
STRIPE_SECRET_KEY=sk_live_...
STRIPE_WEBHOOK_SECRET=whsec_...

# 图片存储（可选）
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
```

### Vercel部署（推荐）

1. **连接GitHub**
   - 将代码推送到GitHub
   - 在Vercel导入项目

2. **配置环境变量**
   - 在Vercel Dashboard添加所有环境变量
   - 确保生产环境变量正确

3. **部署**
   - 点击Deploy
   - 等待部署完成

4. **配置域名**
   - 添加自定义域名
   - 配置DNS记录

### 其他部署方式

- **Docker**: 容器化部署
- **传统服务器**: Node.js服务器部署

详细部署指南请查看：`DEPLOYMENT_GUIDE.md`

# 手动测试脚本

## 快速测试流程（5分钟）

### 1. 启动项目（30秒）

```bash
npm run dev
```

**检查点：**
- ✅ 终端显示 "Ready in X.Xs"
- ✅ 无错误信息
- ✅ 可以访问 http://localhost:3000

---

### 2. 基础功能测试（2分钟）

#### 2.1 首页测试（30秒）
1. 访问 `http://localhost:3000`
2. 检查：
   - ✅ 页面正常加载
   - ✅ Header 显示在顶部
   - ✅ Footer 显示在底部
   - ✅ 看到产品卡片
   - ✅ 看到主题导航

#### 2.2 导航测试（30秒）
1. 点击 "Themes" 链接
2. 检查：✅ 跳转到 `/themes` 页面
3. 点击浏览器返回按钮
4. 检查：✅ 返回首页

#### 2.3 产品详情测试（30秒）
1. 点击任意产品卡片
2. 检查：✅ 跳转到产品详情页
3. 检查：✅ 看到产品图片、价格、描述

#### 2.4 购物车测试（30秒）
1. 点击 "Add to Cart" 按钮
2. 检查：✅ 显示成功提示
3. 检查：✅ 购物车图标显示数字 "1"
4. 点击购物车图标
5. 检查：✅ 跳转到购物车页面

---

### 3. 多语言测试（1分钟）

1. 点击 Header 右上角的语言选择器（EN）
2. 选择 "ES"（西班牙语）
3. 检查：✅ 文本变为西班牙语（如 "Tienda", "Carrito"）
4. 选择 "FR"（法语）
5. 检查：✅ 文本变为法语（如 "Boutique", "Panier"）
6. 选择 "EN"（英语）
7. 检查：✅ 文本恢复为英语

---

### 4. 多货币测试（1分钟）

1. 访问产品详情页 (`/products/1`)
2. 记录 USD 价格（如 $89.99）
3. 点击货币选择器（USD）
4. 选择 "CAD"
5. 检查：✅ 价格更新为 CAD（如 C$121.49）
6. 选择 "GBP"
7. 检查：✅ 价格更新为 GBP（如 £71.09）
8. 选择 "EUR"
9. 检查：✅ 价格更新为 EUR（如 €82.79）

---

### 5. 响应式测试（30秒）

1. 按 F12 打开开发者工具
2. 点击设备图标切换到移动设备模式
3. 选择 "iPhone 12 Pro"
4. 检查：
   - ✅ 布局适配移动端
   - ✅ 看到汉堡菜单（三条横线）
   - ✅ 点击菜单可以打开/关闭

---

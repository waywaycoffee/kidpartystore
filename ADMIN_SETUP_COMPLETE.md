## 🚀 立即开始使用

### 1. 启动开发服务器

```bash
npm run dev
```

### 2. 访问管理后台

打开浏览器访问：
```
http://localhost:3000/admin
```

### 3. 开始使用

- **查看仪表板**: `/admin`
- **管理产品**: `/admin/products`
- **管理图片**: `/admin/images`
- **查看分析**: `/admin/analytics`

## 📁 文件结构

```
app/
├── admin/                    # 管理后台页面
│   ├── layout.tsx           # 管理后台布局
│   ├── page.tsx             # 仪表板
│   ├── products/            # 产品管理
│   │   ├── page.tsx        # 产品列表
│   │   └── new/            # 添加产品
│   ├── images/              # 图片管理
│   │   └── page.tsx
│   └── analytics/           # 销售分析
│       └── page.tsx
└── api/
    └── admin/               # 管理后台 API
        ├── stats/           # 统计数据
        ├── products/        # 产品 API
        └── analytics/       # 分析数据

components/
└── admin/
    └── AdminLayout.tsx      # 管理后台布局组件

data/                        # 数据存储目录
├── .gitkeep
├── products.json           # 产品数据（自动创建）
└── orders.json            # 订单数据（自动创建）
```
